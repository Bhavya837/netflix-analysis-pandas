# Netflix Data Analysis using Pandas

This project explores the Netflix dataset to uncover insights about movies and TV shows available on the platform.  
It focuses on cleaning, analyzing, and visualizing the data using Python, pandas and google colab.


# Project Overview

The goal of this project is to:
- Clean and preprocess the raw Netflix dataset.
- Analyze patterns such as:
  - Distribution of content types (Movies vs TV Shows)
  - Top genres and countries
  - Trends by release year
  - Ratings distribution
- Visualize the insights using plots.

#  Skills Demonstrated

- Handling missing values
- Working with Google Colab
- Data visualization using Matplotlib
- Version control and project publishing on GitHub
